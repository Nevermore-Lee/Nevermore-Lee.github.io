基本类型字面值

1.整数字面值是 int 类型

int a = 65364
long a = 9999999999; //错，右侧字面值是int类型，但超出int范围

2.byte,short,char 比 int 范围小的整数
可以用范围内的值直接赋值

byte a = 127;//对,右侧是 byte 类型
byte a = 128;//错,右侧是 int 类型

3.浮点数是 double 类型

double a = 3.14;
float a = 3.14;//错，右侧是 double

4.字面值后缀

F   float

    double a = 3D;

    0xff   255

基本类型运算规则

1.运算结果的数据类型
与运算项中范围最大的类型一致

3 / 2   得 1
3d / 2  得 1.5

2.byte,short,char 比 int 范围小的整数
运算时，会先自动转成 int 类型

byte a = 2;
byte b = 3;
byte c = a + b;//错，右侧是int + int，结果是 int

3.整数数运算溢出

int a = Integer.MAX_VALUE;

a = a + 1;//负数最小值

01111111111111111111111111111111  +  1
10000000000000000000000000000000

4.浮点数运算不精确

2-1.9     得 0.1000000000000009
4.35*100  得 434.9999999999999994

5.浮点数特殊值

Infinity

  Double.MAX_VALUE*2

&&  逻辑与,两侧同时是真结果是真，
    任意一侧是假结果是假
    短路与: 左侧是假，右侧被忽略

||  逻辑或,两侧同时是假结果是假，
    任意一侧是真结果是真
    短路或: 左侧是真，右侧被忽略


!   非,真变假，假变真

    可以做相反的判断

    如果“非a”是真，即a是假
    if(!a) {

    }

& 位与:两位同时是1结果是1，任意一位是0结果是0
| 位或:两位同时是0结果是0，任意一位是1结果是1
^ 异或:不同是1，相同是0；
对同一个值异或两次得到原值；
可以对boolean值求异或

~ 求反:1变0，0变1

带符号右移位,
符号位是0，左侧补0；
符号位是1，左侧补1。

不带符号右移位，
不管符号位是0还是1，左侧补0

<< 左移位，右侧补0

++

自增、自减

int a = 10;
//a先自增变成11，打印显示a的原值10
System.out.println(a++);

int a = 10;
//a先自增变成11，打印显示a的新值11
System.out.println(++a);

int a = 10;
int b = a++;//a是11, b是10

int a = 10;
a = a++;//a先自增变成11,再被赋a的原值10

  • 局部变量、成员变量

  • 局部变量
    定义在方法中，或局部代码块中

    *)没有默认值，
  必须手动初始化(第一次赋值,会分配内存空间)

    int a;
    a = 10;
    print(a);

*)局部变量作用域，
  到它定义的代码块结束
*)局部变量作用域内，
  不能重复定义

    void a() {
        int i=10;
        if(...) {
            print(i);
            int j = 100;
            //int i = 1000;
        }
        //print(j);
        int j = 10000;
    }

    定义在类中

    *)成员变量有默认值，自动初始化

    默认值：

    引用类型： null

    )可以被类中所有方法访问 )根据访问范围设置，

    可以在类外面访问

    *)局部变量可以与成员变量同名
