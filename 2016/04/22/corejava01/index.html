jdk

  • java 开发库

  • 包含开发工具、基础类库

  • 配置环境变量

    JAVA_HOME
    CLASSPATH
    PATH

  • JAVA_HOME
    配置 jdk 的安装目录路径
    C:\Program Files (x86)\Java\jdk1.7.0

  • CLASSPATH
    CLASSPATH=.

  • PATH
    是系统中已经存在的变量，
    在其中添加新的路径 jdk目录\bin

    %JAVA_HOME%\bin;%PATH%

eclipse

  • 集成的开发环境
  • 解压缩就能用

  • 基本版、java ee企业版

  • 工作空间 workspace

    存放开发的文件的目录

android sdk

  • 首先安装 “android sdk manager”

  • sdk管理器，用来安装、删除
    开发工具、开发库、扩展工具

  • 环境变量

    ANDROID_SDK_HOME

    配置android sdk安装目录路径
d:\software\android-sdk-windows

  • AVD

    安卓虚拟设备

ADT

  • Android 开发工具

  • 是 eclipse 插件
    使用 eclipse 开发平台，来开发安卓应用

  • 安装

    *)在线安装、离线安装，

    自行“百度”

    *)推荐使用的安装方式：

    1)在 eclipse\dropins\ 目录下
  新建 adt 目录

2)复制 adt 离线安装包中的
  plugins 和 features 目录
  到第一步新建的 adt 目录下

  • 在 eclipse 中配置 android sdk 目录路径

    window->preferences->左侧 Android

android studio

  • 新的 android 开发工具

  • eclipse + adt 开发工具将被淘汰

  • 安装绑定 android sdk 的集成版(1G)

  • 第一次启动时，会联网更新

第一个 java 程序 Hello world

1.新建java项目: day0101_HelloWorld
2.新建类: HelloWorld
3.在类中添加代码：

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world!!!");
    }
}

4.保存(保存时自动编译)
5.执行 ctrl+f11

*) 源码文件编译成“字节码文件”

*.java --> *.class

*) 先启动java虚拟机（JVM）
虚拟机会加载字节码文件，
并执行其中 main() 方法代码

第一个 android 程序 Hello world

  1. 新建 android 项目: day0102_HelloWorld

  2. 执行

    右键点击项目->run as->android application
